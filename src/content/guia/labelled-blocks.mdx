<h1 class="text-white py-8">Labelled Blocks</h1>

Los bloques en Zig son expresiones y pueden recibir etiquetas, que se utilizan para producir valores. Aquí, estamos usando una etiqueta llamada <span class="bg-gray-800 rounded-md px-2 text-green-400">"blk"</span>. Los bloques producen valores, lo que significa que se pueden utilizar en lugar de un valor. El valor de un bloque vacío <span class="bg-gray-800 rounded-md px-2 text-purple-400">\{}</span> es un valor del tipo <span class="bg-gray-800 rounded-md px-2 text-purple-400">void</span>.

<Prism lang="zig" code={`test "labelled blocks" {
    const count = blk: {
        var sum: u32 = 0;
        var i: u32 = 0;
        while (i < 10) : (i += 1) sum += i;
        break :blk sum;
    };
    try expect(count == 45);
    try expect(@TypeOf(count) == u32);
}`}/>

Esto puede verse como equivalente a i++ en C.

<Prism lang="zig" code={`blk: {
    const tmp = i;
    i += 1;
    break :blk tmp;
}`}/>
