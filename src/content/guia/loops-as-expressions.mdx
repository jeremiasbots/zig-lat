<h1 class="text-white py-8">Loops como expresiones</h1>


Al igual que <span class="bg-gray-800 rounded-md px-2 text-purple-400">return</span>, <span class="bg-gray-800 rounded-md px-2 text-purple-400">break</span> acepta un valor. Esto se puede usar para producir un valor desde un bucle. Los bucles en Zig también tienen una rama <span class="bg-gray-800 rounded-md px-2 text-purple-400">else</span>, que se evalúa cuando el bucle no se sale con un <span class="bg-gray-800 rounded-md px-2 text-purple-400">break</span>.

<Prism lang="zig" code={`fn rangeHasNumber(begin: usize, end: usize, number: usize) bool {
    var i = begin;
    return while (i < end) : (i += 1) {
        if (i == number) {
            break true;
        }
    } else false;
}

test "while loop expression" {
    try expect(rangeHasNumber(0, 10, 3));
}`}/>
