<h1 class="text-white py-8">While loops</h1>

El bucle while de Zig tiene tres partes: una condición, un bloque y una expresión de continuar.

Sin una expresión de continuar:

<Prism lang="zig" code={`test "while" {
    var i: u8 = 2;
    while (i < 100) {
        i *= 2;
    }
    try expect(i == 128);
}`}/>

Con una expresión de continuar:

<Prism lang="zig" code={`test "Bucle while con una expresión de continuar." {
    var sum: u8 = 0;
    var i: u8 = 1;
    while (i <= 10) : (i += 1) {
        sum += i;
    }
    try expect(sum == 55);
}`}/>

Con un continue: 

<Prism lang="zig" code={`test "Bucle while con continue" {
    var sum: u8 = 0;
    var i: u8 = 0;
    while (i <= 3) : (i += 1) {
        if (i == 2) continue;
        sum += i;
    }
    try expect(sum == 4);
}`}/>

Con un break: 

<Prism lang="zig" code={`test "Bucle while con break" {
    var sum: u8 = 0;
    var i: u8 = 0;
    while (i <= 3) : (i += 1) {
        if (i == 2) break;
        sum += i;
    }
    try expect(sum == 1);
}`}/>
