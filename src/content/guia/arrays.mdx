<h1 class="text-white py-8">Arrays</h1>

Los arrays se representan como <span class="bg-gray-800 rounded-md px-2 text-purple-400">[N]T</span>, donde <span class="bg-gray-800 rounded-md px-2 text-purple-400">N</span> es el número de elementos en el array y <span class="bg-gray-800 rounded-md px-2 text-purple-400">T</span> es el tipo de esos elementos (es decir, el tipo hijo del array).

Para literales de array, <span class="bg-gray-800 rounded-md px-2 text-purple-400">N</span> puede ser reemplazado por <span class="bg-gray-800 rounded-md px-2 text-purple-400">_</span> para inferir el tamaño del array.


<Prism lang="zig" code={`const a = [4]u8{ 'h', 'o', 'l', 'a',  };
const b = [_]u8{ 'm', 'u', 'n', 'd', 'o' };`}/>

Para obtener el tamaño de un array, simplemente accede al campo <span class="bg-gray-800 rounded-md px-2 text-purple-400">len</span> del array.


<Prism lang="zig" code={`const array = [_]u8{ 'h', 'o', 'l', 'a' };
const length = array.len; // 4`}/>
