<h1 class="text-white py-8">Punteros de varios elementos</h1>

A veces, puedes tener un puntero a un número desconocido de elementos. <span class="bg-gray-800 rounded-md px-2 text-purple-400">[*]T</span> es la solución para esto, que funciona como <span class="bg-gray-800 rounded-md px-2 text-purple-400">*T</span> pero también admite la sintaxis de indexación, aritmética de punteros y fragmentación. A diferencia de <span class="bg-gray-800 rounded-md px-2 text-purple-400">*T</span>, no puede apuntar a un tipo que no tenga un tamaño conocido. <span class="bg-gray-800 rounded-md px-2 text-purple-400">\*T</span> se convierte en <span class="bg-gray-800 rounded-md px-2 text-purple-400">[\*]T</span>.

Estos punteros de varios elementos pueden apuntar a cualquier cantidad de elementos, incluyendo 0 y 1.
